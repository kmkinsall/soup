About

Soup Cooking Assistant is an AI-powered kitchen companion that helps you prepare 8 delicious soup recipes.

  • Ask questions about ingredients and substitutions
  • Get step-by-step cooking guidance
  • Troubleshoot cooking issues in real-time
  • Receive personalized soup recommendations

How to Use

  1. Browse Recipes: Click the "Recipes" button to view all 8 soups
  2. Ask Questions: Type your cooking questions in the chat
  3. Quick Actions: Use the suggestion buttons for common questions
  4. Get Guidance: The AI will provide detailed, step-by-step help
  5. Clear Chat: Use the trash icon to start a fresh conversation
